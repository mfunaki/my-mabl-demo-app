# 指示: mabl Test Creation Agent用プロンプトの作成

mablの「Test Creation Agent（生成AIによるテスト作成機能）」に入力するための、自然言語プロンプトを作成してください。
このプロンプトは、mablのWebインターフェースに入力し、自動的にテストステップを生成させるために使用します。

## 参照ファイル
* `/docs/spec-expense-app.md` (セクション7: デモシナリオ)

## 要件
以下の3つのテストケースについて、mablに入力する具体的な指示文を出力してください。

1. **APIテスト用プロンプト:**
    * エンドポイント `/api/reset` をPOSTメソッドで叩き、ステータス200を確認する手順。

2. **Mobileテスト用プロンプト (Android/iOS):**
    * アプリを起動し、ユーザー `employee` / `employee123` でログイン。
    * タイトル「Conference」、金額「1000」を入力して申請ボタンを押す。
    * リストに「Conference」が表示され、ステータスが「PENDING」であることを確認する手順。

3. **Webテスト用プロンプト:**
    * URL `/login` を開き、 `manager` / `manager123` でログイン。
    * リストから「Conference」というテキストを持つ行を探す。
    * その行にある承認ボタンをクリックする。
    * ステータスが「APPROVED」に変わったことを確認する手順。
